# TareaUnidad4EstrDatos
  # Tarea de Teoróa de la Complejidad Algorítmica y Estructuras de Datos - Unidad 4

Este repositorio contiene las soluciones desarrolladas para la tarea de la Unidad 4, enfocadas en el análisis asintótico y la eficiencia de algoritmos de ordenamiento y búsqueda.

## Contenido del Proyecto
* **MatrizCompleja.java**: Implementación de una matriz de 1000x1000 (1 millón de elementos) para evaluar el rendimiento práctico de algoritmos de ordenamiento ($Merge$, $Shell$, $Counting$, $Radix$) y algoritmos de búsqueda (Secuencial, Binaria e Interpolación).
* **AnalisisEficienciaSuma.java**: Solución al problema de detección de duplas de suma, contrastando una solución genérica $O(K^2)$ frente a una optimizada de dos punteros con complejidad lineal $O(K)$.

## Tecnologías Utilizadas
* **Lenguaje:** Java 
* **IDE:** Eclipse IDE
* **Entorno de Ejecución:** OpenJDK 26
